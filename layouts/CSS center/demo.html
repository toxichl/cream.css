文字垂直居中

  • 非常简单，确保父元素 height = line-height，且 text-align: center;

  • 注意，一旦字符数太多，这种方法就会出现问题，就像这样:
toxichltoxichltoxichltoxichl
  • 可以设置overflow:hidden来隐藏，像这样：
toxichltoxichltoxichltoxichl
  • 总感觉不够优雅，所以说，记住这种方式的缺点，不能存在多行文字
  • 因为这种方式设置了line-height，所以即使设置 word-break: break-all，也一定会溢出到下一行

文字垂直居中 - 多行文本解决方案 - table

  • 设置父元素： display: table, 设置子元素：display: table-cell
  • 父元素要定宽高，子元素宽高不用设置，会自动扩宽到父元素的宽高
  • 子元素必须设置word-break: break-all;，谨记！！

块状元素居中，关键点：

  • 需要定父元素的高度，因为用到了line-height
  • 需要居中块状元素设置为inline-block元素，然后用vertical-align: middle;配合父元素的text-align: center;和line-height来实现

块状元素居中 - 最粗暴也是最简单的方式

  • 需要居中的元素定位为 position: absolute;
  • 注意，相应的，父级元素要设置为 position: relative;
  • 同时设置子元素:top:0; left:0; right:0; bottom:0; margin: auto;

块状元素居中 - flexbox

  • 非常简单，设置父元素： display: flex; align-items: center; justify-content: center;即可
